### 4. 事件模型

**实现文件**: `src/notifier/dispatcher.py:21-43`

```python
class NotifyLevel(str, enum.Enum):
    P0 = "P0"       # 系统崩溃/数据丢失
    P1 = "P1"       # 研究失败/模型异常
    P2 = "P2"       # 低优先级预警
    SIGNAL = "SIGNAL"     # 信号审批/触发
    RESEARCH = "RESEARCH" # 研究完成摘要
    NOTIFY = "NOTIFY"     # 系统通知/日报

@dataclass
class DecisionEvent:
    event_type: str          # RESEARCH | SIGNAL | STRATEGY | DAILY | SYSTEM
    notify_level: NotifyLevel
    event_code: str
    title: str
    body: str                # Markdown 正文
    strategy_id: str = ""
    model_id: str = ""
    signal_id: str = ""
    session_id: str = ""
    trace_id: str = ""
    extra: dict = field(default_factory=dict)
```

**状态**: ✅ 已实现，支持所有必需字段
