## 🎯 核心指标标准（激进版）

### 1. Sharpe Ratio（夏普比率）

**目标推导**：
- 年化收益：50%-80%（激进目标）
- 假设无风险利率：3%
- 年化波动率目标：≤ 20%（适度放宽）
- **目标 Sharpe**：(65% - 3%) / 20% = **3.1**

**评分标准**：

| Sharpe Ratio | 评分 | 等级 | 说明 |
|--------------|------|------|------|
| ≥ 3.0 | 30/30 | S | 优秀，超出目标 |
| 2.5 - 3.0 | 25/30 | A | 良好，接近目标 |
| 2.0 - 2.5 | 20/30 | B | 合格，需优化 |
| 1.5 - 2.0 | 15/30 | C | 勉强，需大幅优化 |
| < 1.5 | 0/30 | D | 不合格 |

**计算公式**：
```
Sharpe Ratio = (年化收益率 - 无风险利率) / 年化波动率
```

---

### 2. 最大回撤（Max Drawdown）

**目标推导**：
- 月回撤目标：1%-1.5%
- 极限容忍：2%
- **目标最大回撤**：≤ 1.5%

**评分标准**：

| 最大回撤 | 评分 | 等级 | 说明 |
|---------|------|------|------|
| ≤ 1.0% | 30/30 | S | 优秀，风控极佳 |
| 1.0% - 1.5% | 25/30 | A | 良好，符合目标 |
| 1.5% - 2.0% | 20/30 | B | 合格，接近极限 |
| 2.0% - 3.0% | 10/30 | C | 勉强，风险较高 |
| > 3.0% | 0/30 | D | 不合格，风险过高 |

**计算公式**：
```
最大回撤 = (峰值净值 - 谷值净值) / 峰值净值
```

---

## 🔧 调优策略

### 针对不同问题的调优方向

#### 问题 1：Sharpe Ratio 低

**原因**：收益不足或波动过大

**调优方向**：
1. 提高入场精度（优化因子权重）
2. 优化止盈止损比例
3. 降低交易频率（减少噪音交易）

---

#### 问题 2：最大回撤过大

**原因**：风控不足或连续亏损

**调优方向**：
1. 降低单品种仓位（10% → 5%）
2. 收紧止损条件
3. 增加相关性过滤（避免同时持有高相关品种）
4. 增加市场环境过滤（避免极端行情）

---

#### 问题 3：胜率低

**原因**：入场时机不佳或止损过紧

**调优方向**：
1. 提高入场阈值（0.55 → 0.60）
2. 增加确认信号（多因子共振）
3. 放宽止损（但要控制单笔亏损）

---

#### 问题 4：盈亏比低

**原因**：止盈过早或止损过晚

**调优方向**：
1. 优化止盈策略（移动止盈）
2. 收紧止损
3. 增加趋势跟踪（让利润奔跑）

3. **胜率 ≥ 55%**
4. **盈亏比 ≥ 2.0**

### 风控标准

1. **单品种仓位 ≤ 10%**（保守 5%）

3. **单品种熔断 ≤ 1.0%**

3. **风控严格**：多层风控，层层保护
4. **逐步放开**：从保守到激进，逐步验证
