# 研究员 24/7 重构 - 实现文档

## 概述

本文档记录 TASK-0121 研究员 24/7 重构的完整实现。

## 架构设计

### 多进程架构

系统采用 5 进程并行架构：

1. **KlineMonitor** - K 线监控器
   - 实时监控 35 个期货品种
   - 检测异常波动（> 2%）
   - 推送异常事件到共享队列

2. **NewsCrawler** - 新闻爬虫
   - 爬取新浪财经、东方财富、金十数据等
   - 过滤期货相关新闻
   - 增量去重

3. **FundamentalCrawler** - 基本面爬虫
   - 爬取库存、产量、宏观数据
   - 定时采集
   - 增量去重

4. **LLMAnalyzer** - LLM 分析器
   - 从共享队列消费数据
   - 调用 qwen3:14b 分析
   - 输出到报告缓冲区

5. **ReportGenerator** - 报告生成器
   - 定时汇总分析结果
   - 生成 JSON 和 Markdown 报告
   - 推送到 Mini、飞书、邮件

### 数据流

```
Mini API (K线数据)
    ↓
KlineMonitor → 共享队列
    ↓
NewsCrawler → 共享队列
    ↓
FundamentalCrawler → 共享队列
    ↓
LLMAnalyzer (qwen3:14b) → 报告缓冲区
    ↓
ReportGenerator → Mini API / 飞书 / 邮件
```
