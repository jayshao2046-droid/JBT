## 问题描述

当前策略调优流水线只能生成满足最低标准的策略（交易次数≥10, Sharpe≥0.5），但无法达到生产级标准。需要实现完整的迭代优化流程，通过多轮 Researcher ↔ Critic 迭代，将策略优化到可生产水平。

## 目标

**生产级标准**（OOS 期间）：
- Sharpe Ratio ≥ 1.5
- 交易次数 ≥ 20
- 胜率 ≥ 50%
- 最大回撤 ≤ 3%
- 年化收益 ≥ 15%

**迭代策略**：
- 第 1-3 轮：快速迭代，达到最低标准（交易次数≥10, Sharpe≥0.5）
- 第 4-10 轮：中等迭代，逐步提升到中等水平（交易次数≥15, Sharpe≥1.0）
- 第 11-30 轮：精细迭代，冲刺生产级标准（Sharpe≥1.5, 交易次数≥20）

### 2. 三阶段迭代策略

#### 阶段 1：快速达标（1-3 轮）
**目标**: 交易次数≥10, Sharpe≥0.5
**策略**: 
- 放宽市场过滤器（ATR/ADX/volume_ratio）
- 调整信号阈值（RSI/MACD）
- 如果 3 轮未达标，标记失败

#### 阶段 2：中等提升（4-10 轮）
**目标**: 交易次数≥15, Sharpe≥1.0, 胜率≥45%
**策略**:
- 优化入场条件（提高信号质量）
- 调整止损止盈（改善盈亏比）
- 引入确认机制（减少假信号）

#### 阶段 3：生产冲刺（11-30 轮）
**目标**: Sharpe≥1.5, 交易次数≥20, 胜率≥50%
**策略**:
- 精细调整参数组合
- 优化风险控制
- 平衡交易频率与质量

### 4. 参数调整逻辑

```python
def _apply_critic_suggestions(strategy: dict, critic_json: dict) -> dict:
    """根据 Critic 建议调整策略参数"""
    changes = critic_json.get("solution", {}).get("changes", {})
    
    # 更新因子参数
    for factor in strategy.get("factors", []):
        factor_name = factor.get("factor_name", "")
        params = factor.get("params", {})
        
        for key, value in changes.items():
            # 匹配参数名（支持模糊匹配）
            if key in params or key.lower() in [k.lower() for k in params.keys()]:
                params[key] = float(value)
        
        factor["params"] = params
    
    # 更新市场过滤器
    market_filter = strategy.get("market_filter", {})
    for condition in market_filter.get("conditions", []):
        for key, value in changes.items():
            if key in condition:
                condition[key] = float(value)
    
    return strategy
```
