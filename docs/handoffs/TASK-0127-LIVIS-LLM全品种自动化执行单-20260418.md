## 5. 生产标准

可纳入生产候选的最低结论：

- 评分 ≥ 80
- Sharpe ≥ 2.5
- 最大回撤 ≤ 1.5%
- 胜率 ≥ 55%
- 盈亏比 ≥ 2.0

同时必须满足流水线硬标准：

- OOS Sharpe ≥ 1.5
- OOS 交易次数 ≥ 20
- OOS 胜率 ≥ 50%
- OOS 最大回撤 ≤ 2%
- IS/OOS Sharpe 比 ≤ 2.0

## 6. TqSdk 兼容底线

每个 YAML 至少检查：

1. 含 `transaction_costs`
2. 止损止盈为 ATR 配置
3. `risk.force_close_day: "14:55"`
4. 夜盘品种含 `risk.force_close_night`
5. `no_overnight: true`
6. 主力合约代码，不用指数合约
7. signal 不引入模板外变量
