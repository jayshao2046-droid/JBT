新架构（D115 unified_strategy_runner.py）：**
```
单一 TqSim(200,000元) + 全品种 K 线订阅（1个 TqApi 实例）

策略信号工作线程（N个线程，共享 K 线，只读）
  FC-New_34_metals_v1  → [rb, hc, i, j, jm]
  FC-New_34_Oil_v1     → [y, p, OI, m]
  FC-New_34_chemicals_v1 → [TA, MA, v, pp, ru]
  FC-New_34_textile_v1 → [CF]
  FC-New_33_rb_v1      → [rb]   ← 品种专属（更细粒度参数）
  ... 全部 enabled FC
   ↓ 每10分钟：(fc_id, symbol, direction, confidence, score) → signals_queue

信号聚合器（主线程，每10分钟）
  同品种多FC → 取最高 score（confidence × sharpe_weight）
  全局按 score 降序排名
  置信度分层映射手数（≥0.90→3手, ≥0.80→2手, ≥0.65→1手）
  限制 max_concurrent_open 个品种

TqSim 执行层（单一账户，全局风控）
  动态开仓 / 平仓（信号消失/方向反转）
  全局日亏熔断 3000元 / 品种熔断 600元
  14:55日盘强平 / 22:55夜盘强平
```

### 置信度分层（capital_allocation.yaml）

| 置信度 | 标签 | 手数 |
|--------|------|------|
| ≥ 0.90 | 超高置信度 | 3手 |
| ≥ 0.80 | 高置信度 | 2手 |
| ≥ 0.65 | 中置信度 | 1手 |
| ≥ 0.55 | 低置信度 | 1手 |
| < 0.55 | 拒绝 | 0手 |

Sharpe 权重：sharpe≤0 → 0.1，sharpe≥2.0 → 2.0，其余取原值（最小 0.1）
