## 📦 已交付模块

### 1. L1/L2/L3 门控通知模板
**文件**: `services/decision/src/notifier/gate_notifications.py`

**功能**:
- L1 快速门控通知（phi4-reasoning 风险标记）
- L2 深度审查通知（多维度风险评估）
- L3 在线确认通知（人工确认/拒绝/超时）
- 去重策略：60秒内同 signal_id + 同阶段只发一次
- P0/P1 二次升级：30秒重试 + 邮件兜底

**关键方法**:
```python
GateNotificationHelper.create_l1_notification(...)
GateNotificationHelper.create_l2_notification(...)
GateNotificationHelper.create_l3_notification(...)
```

### 2. 每日汇总邮件
**文件**: `services/decision/src/notifier/daily_summary.py`

**功能**:
- L1/L2/L3 总量与通过率统计
- 按品种命中率 Top 10
- 拒绝原因 Top 5（L1/L2/L3 分层）
- 失败通知统计（飞书/邮件/双通道）
- 风险摘要（严重/警告分级）

**使用方式**:
```python
from notifier.daily_summary import get_daily_summary, send_daily_summary

# 记录 L1 结果
summary = get_daily_summary()
summary.record_l1(symbol="IF2505", passed=True)

# 每日 23:55 发送汇总
send_daily_summary(summary)
```

---

### 3. 健康监控系统
**文件**: `services/decision/src/notifier/health_monitor.py`

**功能**:
- 5 大组件监控：数据源、模型服务、策略引擎、通知系统、门控审查器
- 三级状态：Healthy / Degraded / Critical
- 15 分钟健康快照（保留 24 小时历史）
- 每日健康报告（状态 + 24 小时趋势）
- 智能通知：仅在 Degraded/Critical 时发送快照

**使用方式**:
```python
from notifier.health_monitor import get_health_monitor

monitor = get_health_monitor()

# 每 15 分钟执行
snapshot = monitor.take_snapshot()
monitor.send_snapshot_notification(snapshot)

# 每日 08:00 发送报告
monitor.send_daily_report()
```
