## 七、API 认证与安全

### 7.1 API Key 认证

所有服务均使用 `X-API-Key` Header 认证：

```typescript
headers: {
  "X-API-Key": process.env.BACKTEST_API_KEY,
  "Content-Type": "application/json"
}
```

环境变量：
- `DATA_API_KEY` - Data 服务
- `DECISION_API_KEY` - Decision 服务
- `BACKTEST_API_KEY` - Backtest 服务
- `SIM_API_KEY` - Sim-Trading 服务

### 7.2 公开路径

以下路径无需认证：
- `/health`
- `/api/v1/health`
- `/api/v1/version`
- `/ready`

### 7.3 CORS 配置

所有服务均配置 CORS：
- 允许来源：`http://localhost:3001`, `http://localhost:3005`
- 允许方法：`GET`, `POST`, `PUT`, `DELETE`, `OPTIONS`
- 允许头部：`Content-Type`, `Authorization`, `X-API-Key`
