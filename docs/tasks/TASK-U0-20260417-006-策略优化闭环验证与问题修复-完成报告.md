#### 2.3.3 关键修复：波动率计算

**问题**:
- 修复前：使用 `ATR / Close`，结果偏低 20-40 倍
- 螺纹钢实测：0.06% vs 行业基准 2-4%

**修复**:
```python
def _calculate_volatility(self, bars: list[dict]) -> float:
    """计算波动率（年化标准差）
    
    使用日收益率标准差 * sqrt(252*4) 计算年化波动率。
    这比 ATR/Close 更准确，因为 ATR 受 K 线周期影响。
    """
    # 计算收益率
    returns = [(closes[i] - closes[i-1]) / closes[i-1] for i in range(1, len(closes))]
    
    # 计算标准差
    std_dev = sqrt(variance(returns))
    
    # 年化波动率（60分钟K线，252交易日）
    annualization_factor = sqrt(252 * 4)
    
    return std_dev * annualization_factor
```
