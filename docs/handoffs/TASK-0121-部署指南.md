# 研究员 24/7 重构 - 部署指南

## 环境要求

- Python 3.10+
- Ollama (qwen3:14b 模型)
- Mini 数据服务 (192.168.31.156:8105)
- 万兆内网连接

## 依赖安装

```bash
cd /Users/jayshao/JBT/services/data

# 安装依赖
pip install -r requirements.txt

# 额外依赖
pip install beautifulsoup4 requests
```

## 配置

### 1. 数据源配置

编辑 `configs/researcher_sources.yaml`：

```yaml
news_sources:
  - name: "新浪财经"
    url: "https://finance.sina.com.cn/futuremarket/"
    interval: 300
```

### 2. Mini API 配置

确保 Mini 数据服务运行：

```bash
# 在 Mini (192.168.31.156) 上
docker ps | grep jbt-data
```

### 3. Ollama 配置

确保 Ollama 运行并加载 qwen3:14b：

```bash
# 在 Alienware 上
ollama list
ollama run qwen3:14b
```

## 启动

### 单进程测试

```bash
# 测试 K 线监控器
python -c "from researcher.kline_monitor import KlineMonitor; import multiprocessing as mp; m = KlineMonitor(mp.Queue(), mp.Event()); print('OK')"

# 测试新闻爬虫
python -c "from researcher.news_crawler import NewsCrawler; import multiprocessing as mp; c = NewsCrawler(mp.Queue(), mp.Event()); print('OK')"
```

### 多进程启动

```bash
# 启动研究员（前台）
python -m researcher.process_manager

# 启动研究员（后台）
nohup python -m researcher.process_manager > /data/researcher/logs/researcher.log 2>&1 &
```
