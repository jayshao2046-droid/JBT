## 1. 背景与动机

根据 2026-04-15 风控常规体检报告（`docs/reports/20260415-风控常规体检报告.md`），发现：

- **4 个 P0 高危漏洞**：API 认证绕过、命令注入、LLM Prompt 注入、敏感信息泄露
- **7 个 P1 中危漏洞**：飞书 Token 泄露、内存泄漏、HTML 注入、文件权限、Token 文件读取不安全、XPath 注入、JSON 解析异常
- **6 个 P2 低危问题**：硬编码超时、缺少速率限制、并发安全、缺少审计日志等
