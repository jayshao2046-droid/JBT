   这个脚本会自动：

5. **查看日志**

## 🔍 故障排查

### Alienware 服务启动失败

**检查日志**：
```powershell
Get-Content C:\Users\17621\jbt\runtime\researcher\logs\server.log -Tail 20
```

**常见问题**：
- Python 依赖缺失：`pip install -r requirements.txt`
- 端口被占用：检查 8199 端口
- 权限不足：以管理员身份运行

### Studio 服务启动失败

**检查日志**：
```bash
tail -50 ~/jbt/services/decision/logs/decision.log
```

**常见问题**：
- 端口被占用：`lsof -i :8104`
- Python 依赖缺失：`pip install -r requirements.txt`

### 队列为空

**检查报告目录**：
```powershell
Get-ChildItem D:\researcher_reports -Recurse
```

**手动触发**：
```powershell
$hour = (Get-Date).Hour
Invoke-RestMethod -Uri "http://localhost:8199/run?hour=$hour" -Method Post
```

### Studio 无法读取报告

**测试网络连通性**：
```bash
curl http://192.168.31.223:8199/health
curl http://192.168.31.223:8199/reports/latest
```

## 🎯 关键日志关键词
