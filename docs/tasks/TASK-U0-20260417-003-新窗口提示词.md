## 当前问题
**KeyError: 'MACD'** - qwen-coder 生成的代码仍在尝试访问 `bars["MACD"]`，而不是使用 `ta_macd()` 函数。

### 根因
yaml_signal_executor.py 的 researcher prompt 没有明确强调：
1. bars 只有 6 个字段：datetime, open, high, low, close, volume
2. 必须使用 ta_* 函数计算技术指标（ta_macd, ta_rsi, ta_ema 等）
3. 不能直接访问 bars["MACD"] 或 bars["RSI"]

## 可用的 ta_* 函数
```python
ta_ema(values, period)           # 指数移动平均
ta_sma(values, period)           # 简单移动平均
ta_rsi(values, period)           # 相对强弱指标
ta_macd(values, fast, slow, signal)  # MACD
ta_atr(highs, lows, closes, period)  # 平均真实波幅
ta_adx(highs, lows, closes, period)  # 平均趋向指标
ta_bollinger(values, period, std_dev)  # 布林带
ta_volume_ratio(volumes, period)     # 成交量比率
ta_cci(highs, lows, closes, period)  # 商品通道指标
ta_williams_r(highs, lows, closes, period)  # 威廉指标
ta_kdj(highs, lows, closes, period, k_period, d_period)  # KDJ
ta_obv(closes, volumes)          # 能量潮
```
