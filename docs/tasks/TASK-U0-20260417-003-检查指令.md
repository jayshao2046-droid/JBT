# Studio 调优问题检查指令

## 执行步骤

### 步骤 1：复制检查脚本到 Studio

在你的 **MacBook Air** 上执行：

```bash
# 复制状态检查脚本
scp /tmp/check_studio_status.sh jay@192.168.31.187:~/check_studio_status.sh

# 复制报告检查脚本
scp /tmp/check_studio_reports.sh jay@192.168.31.187:~/check_studio_reports.sh
```

### 步骤 2：登录 Studio 并执行检查

```bash
# 登录 Studio
ssh jay@192.168.31.187

# 赋予执行权限
chmod +x ~/check_studio_status.sh
chmod +x ~/check_studio_reports.sh

# 执行状态检查
./check_studio_status.sh

# 执行报告检查
./check_studio_reports.sh
```

### 步骤 3：手动快速检查（如果脚本失败）

如果上述脚本执行失败，可以手动执行以下命令：

```bash
# 1. 检查代码版本
cd ~/JBT
git log -1 --oneline services/decision/src/research/yaml_signal_executor.py

# 2. 检查是否有 ta_* 函数
grep "def ta_ema" services/decision/src/research/yaml_signal_executor.py
grep "def ta_macd" services/decision/src/research/yaml_signal_executor.py

# 3. 检查 Ollama 服务
curl -s http://192.168.31.142:11434/api/tags | head -30

# 4. 检查 Mini data API
curl -s http://192.168.31.76:8105/api/v1/symbols | head -20

# 5. 查看最近的报告
ls -lt ~/JBT/services/decision/reports/tuning/ | head -10

# 6. 查看具体报告内容（如果存在）
cat ~/JBT/services/decision/reports/tuning/STRAT_p_trend_120m_001_report.json | head -50
```
