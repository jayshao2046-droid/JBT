## 三、需要做的调整和修改

### 3.1 项目结构调整

#### 新建统一门户项目

```bash
# 创建新项目
mkdir -p services/portal
cd services/portal
pnpm create next-app@latest . --typescript --tailwind --app --use-pnpm
```

#### 迁移策略

**方案 A：全量迁移（推荐）**
- 把四个 `*_web` 的所有页面和组件迁移到 `services/portal`
- 保持原有目录结构，只是放在统一项目下
- 优点：完全统一，易于维护
- 缺点：迁移工作量大

**方案 B：渐进式迁移**
- 先建立 portal 框架（登录、首页、设置）
- 四个临时看板通过 iframe 嵌入
- 逐步迁移页面到 portal
- 优点：快速上线
- 缺点：过渡期维护复杂

**建议**：采用方案 A，一次性迁移完成。

### 3.7 认证和权限控制

#### 新增认证中间件

```typescript
// portal/middleware.ts
import { NextResponse } from 'next/server'
import type { NextRequest } from 'next/server'

export function middleware(request: NextRequest) {
  const token = request.cookies.get('auth-token')
  
  // 公开路径
  const publicPaths = ['/login', '/register']
  if (publicPaths.includes(request.nextUrl.pathname)) {
    return NextResponse.next()
  }
  
  // 未登录，重定向到登录页
  if (!token) {
    return NextResponse.redirect(new URL('/login', request.url))
  }
  
  // 已登录，检查权限
  const user = verifyToken(token.value)
  const path = request.nextUrl.pathname
  
  // 权限检查
  if (!hasPermission(user, path)) {
    return NextResponse.redirect(new URL('/403', request.url))
  }
  
  return NextResponse.next()
}

export const config = {
  matcher: ['/((?!api|_next/static|_next/image|favicon.ico).*)'],
}
```

#### 新增认证 API

```typescript
// portal/lib/auth.ts
export async function login(username: string, password: string) {
  const res = await fetch('/api/portal/auth/login', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ username, password }),
  })
  
  if (!res.ok) throw new Error('登录失败')
  
  const { token, user } = await res.json()
  
  // 保存 token 到 cookie
  document.cookie = `auth-token=${token}; path=/; max-age=86400`
  
  return user
}

export async function logout() {
  document.cookie = 'auth-token=; path=/; max-age=0'
  window.location.href = '/login'
}

export function getCurrentUser() {
  // 从 cookie 读取 token 并解析
  const token = getCookie('auth-token')
  if (!token) return null
  
  return verifyToken(token)
}
```

**调整内容**：
- 新增登录页
- 新增认证中间件
- 新增权限检查逻辑
- 所有页面自动受保护

from fastapi.middleware.cors import CORSMiddleware

app = FastAPI(title="JBT Portal API")

    CORSMiddleware,
    allow_origins=["http://localhost:3000"],

    allow_methods=["*"],
