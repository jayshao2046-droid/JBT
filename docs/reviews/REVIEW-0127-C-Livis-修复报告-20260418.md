## 一、修复内容

### ✅ P3 修复 — CZCE 品种交易所前缀硬编码

**问题**：`run_full_pipeline_35_symbols.py` 第 317、379 行硬编码 `DCE.{symbol}`，导致 CZCE 品种（cf/sr/ma/ta/ap/eg 等）生成错误前缀。

**修复位置**：`services/decision/scripts/run_full_pipeline_35_symbols.py` 第 315-322 行

**修复方案**：
1. 新增 `CZCE_COMMODITIES` 集合（15 个品种）
2. 根据品种判断交易所前缀：
   - CZCE 品种 → `CZCE.{symbol.upper()}`（如 `CZCE.AP511`）
   - 其他品种 → `DCE.{symbol}`（如 `DCE.p2505`）
3. 统一使用 `symbol_with_exchange` 变量传递给 `calculate_features()` 和 `generate_yaml_strategy()`

**修复代码**：
```python
# CZCE 品种列表
CZCE_COMMODITIES = {"cf", "sr", "ma", "ta", "ap", "cy", "fg", "oi", "rm", "rs", "sf", "sm", "wh", "zc", "eg"}

# 确定交易所前缀
exchange_prefix = "CZCE" if symbol.lower() in CZCE_COMMODITIES else "DCE"
symbol_with_exchange = f"{exchange_prefix}.{symbol.upper() if exchange_prefix == 'CZCE' else symbol}"

# 1. 品种特征分析
features = await profiler.calculate_features(symbol=symbol_with_exchange)

# 生成 YAML
yaml_path = await generator.generate_yaml_strategy(
    strategy_design=design,
    symbol=symbol_with_exchange,  # 使用正确的交易所前缀
    timeframe=template["timeframe"],
)
```

**验证**：
- CZCE 品种（如 ap）生成 `CZCE.AP511`
- DCE 品种（如 p）生成 `DCE.p2505`
- 其他品种（如 rb）生成 `DCE.rb2505`

### ✅ P4 修复 — `_preprocess_yaml()` async/sync 混用

**问题**：`tqsdk_backtest_client.py` 第 303-307 行在同步方法中使用 `loop.run_until_complete()`，导致 async 上下文中抛出 `RuntimeError: This event loop is already running`。

**修复位置**：`services/decision/src/research/tqsdk_backtest_client.py` 第 283-318 行

**修复方案**：
1. 将 `_preprocess_yaml()` 改为 `async def`
2. 删除 `import asyncio` 和 `loop.run_until_complete()` 逻辑
3. 直接 `await self._contract_resolver.get_main_contract(symbol)`
4. 调用处（第 104 行）改为 `await self._preprocess_yaml(strategy)`

**修复代码**：
```python
# 方法签名改为 async
async def _preprocess_yaml(self, strategy: dict) -> dict:
    ...
    for symbol in symbols:
        if symbol.endswith(".0") or symbol.endswith("0"):
            try:
                # 直接 await，不再使用 run_until_complete
                main_contract = await self._contract_resolver.get_main_contract(symbol)
                logger.info(f"主力合约转换: {symbol} → {main_contract}")
                converted_symbols.append(main_contract)
            except Exception as e:
                logger.warning(f"主力合约转换失败 ({symbol}): {e}，保持原值")
                converted_symbols.append(symbol)
    ...

# 调用处（submit_backtest 方法）
strategy = await self._preprocess_yaml(strategy)
```

**验证**：
- 不再抛出 `RuntimeError: This event loop is already running`
- 合约转换逻辑正常执行（`DCE.p0` → `DCE.p2505`）
