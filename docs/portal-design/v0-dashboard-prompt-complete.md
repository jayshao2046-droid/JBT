## 交互功能要求

1. **一键平仓按钮**：点击后弹出确认对话框，显示当前持仓汇总和风险提示
2. **策略信号操作**：点击"执行"按钮后显示交易确认对话框
3. **持仓详情**：点击持仓行可展开显示更多信息（成本分布、历史操作等）
4. **数据刷新**：所有数据每30秒自动刷新一次（模拟）
