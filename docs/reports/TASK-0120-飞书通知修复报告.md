## 问题描述

研究员通知发送到飞书资讯群失败，返回错误码 `19024: Key Words Not Found`

## 根因分析

资讯群 webhook 配置了关键词白名单：`"BotQuant 资讯"`  
但 `notifier.py` 三处飞书卡片的 note 字段中缺少该关键词

## 修复内容

**文件**: `services/data/src/researcher/notifier.py`

在三处卡片的 note 开头添加 `"BotQuant 资讯 | "`：

1. **165行** - 研究员报告卡片（blue 模板）
2. **263行** - 执行失败告警卡片（orange 模板）  
3. **312行** - 突发紧急事件卡片（red 模板）

**修改前**:
```python
"content": f"JBT 数据研究员 | {hour}:{minute} | ..."
```

**修改后**:
```python
"content": f"BotQuant 资讯 | JBT 数据研究员 | {hour}:{minute} | ..."
```

✅ 三种通知类型全部测试通过：
- 研究员报告通知
- 执行失败告警
