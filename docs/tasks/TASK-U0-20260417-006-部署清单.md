1. **停止 Alienware 服务**
   ```powershell

   ```

2. **回滚 decision 评级器**
   ```bash

   # model: str = "qwen3:14b" → model: str = "phi4-reasoning:14b"
   ```

3. **重启 decision 服务**
