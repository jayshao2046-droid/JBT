# Drafts for sim-trading contracts

本目录用于放置 `sim-trading` 的契约草稿，供项目架构师在获得 P0 Token 前准备与自校验。

流程：

1. 在本目录中撰写并完善 `order.md`、`position.md`、`account.md`、`api.md` 草稿。
2. 草稿自校验无误后，项目架构师向 Jay.S 申请 P0 Token（通过 lockctl 生成并粘贴）。
3. 收到实际 P0 Token 后，项目架构师将经终审同意的草稿移动到 `shared/contracts/sim-trading/`（P0 保护区），并在 `docs/locks/TASK-0002-lock.md` 中登记提交 ID 与锁回信息。

注意：草稿区仅为临时准备与讨论之用，禁止放置可执行代码或业务逻辑实现。
