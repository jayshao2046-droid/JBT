### 4.2 P2 问题（影响开发体验）

#### 问题 2：Python 3.9 类型注解兼容性

**问题描述**：
```
TypeError: unsupported operand type(s) for |: '_GenericAlias' and 'type'
```

**影响文件**：
- tests/publish/test_strategy_importer.py
- tests/test_api_auth.py
- tests/test_health.py
- tests/test_publish.py
- tests/test_strategy.py

**原因**：使用了 Python 3.10+ 的 `str | None` 语法，但运行环境是 Python 3.9

**修复建议**：
```python
# 错误写法（Python 3.10+）
def func(param: str | None) -> dict | None:
    pass

# 正确写法（Python 3.9 兼容）
from typing import Optional
def func(param: Optional[str]) -> Optional[dict]:
    pass
```

**优先级**：P2（影响开发体验，但不阻塞生产）

---

#### 问题 3：集成测试模块依赖缺失

**问题描述**：
```
ModuleNotFoundError: No module named 'decision'
File: tests/test_researcher_integration.py
Import: from decision.src.llm.researcher_loader import ResearcherLoader
```

**影响**：集成测试无法运行

**修复建议**：
```python
# 修改导入路径
# 错误：from decision.src.llm.researcher_loader import ResearcherLoader
# 正确：from src.llm.researcher_loader import ResearcherLoader
```

**优先级**：P2（仅影响集成测试）
