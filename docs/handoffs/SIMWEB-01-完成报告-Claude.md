### 5.4 P1-1：交易绩效 KPI（7个指标）

**实现方式**：
- 后端：新建 `src/stats/performance.py`（122行）

**指标清单**：
1. 胜率：盈利订单数 / 总订单数
2. 盈亏比：平均盈利 / 平均亏损

5. 今日盈亏：当日累计盈亏
6. 本周盈亏：本周累计盈亏
7. 本月盈亏：本月累计盈亏

## 六、已知问题与待完成事项

### 6.1 待完成事项

1. **音频文件**：需要添加真实的 `public/alert.mp3` 文件（当前为占位符）
2. **实时数据对接**：K线数据和市场异动当前为模拟数据，需要对接真实数据源

4. **SSE推送优化**：风控告警SSE当前为模拟推送，需要对接真实告警队列

### 6.2 已知限制

1. **浏览器兼容性**：仅支持现代浏览器（Chrome/Safari/Edge）
2. **通知权限**：浏览器通知需要用户授权
3. **音频播放**：可能被浏览器阻止（需要用户交互后才能播放）
4. **权益历史数据**：只保留最近30天数据（超过30天自动清理）
