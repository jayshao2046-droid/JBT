## 功能需求

```python
# strategy_importer.py
class StrategyImporter:
    def import_from_yaml(self, path: str) -> StrategyModel: ...
    def import_from_dict(self, data: dict) -> StrategyModel: ...

# yaml_importer.py
def load_strategy_yaml(path: str) -> dict: ...
def validate_strategy_schema(data: dict) -> bool: ...
```

策略 YAML 最小结构（与现有 model 层保持兼容）：
```yaml
name: string
symbol: string
exchange: string
direction: long/short
entry_rules: [...]
exit_rules: [...]
risk_params: {...}
```

## 验收标准

1. `StrategyImporter.import_from_yaml(path)` 能正确解析 YAML 并返回 StrategyModel。
2. 格式非法时抛出 `ValueError`，文件不存在时抛出 `FileNotFoundError`。
3. 新建文件有单元测试覆盖（happy path + 边界）。
4. 无跨服务 import，无硬编码路径。
5. `model/` 层现有逻辑无回归。
