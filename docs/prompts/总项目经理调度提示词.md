## 2.1 当前正式派发任务

目标 Agent：项目架构师

任务名称：TASK-0002 阶段一继续执行（契约草稿完善 + 自校验）

任务要求：

1. 读取 `docs/handoffs/TASK-0002-架构师阶段一继续执行.md`，按交接单继续工作。
2. 核对并修正共享状态文件中的重复或过时表述，尤其是 `docs/prompts/公共项目提示词.md` 当前存在的重复阶段描述与待审状态残留。
3. 在 `shared/contracts/drafts/sim-trading/` 完成四份契约草稿：`order.md`、`position.md`、`account.md`、`api.md`。
4. 草稿内容只允许提取最小必要字段与接口说明，禁止复制 legacy 业务逻辑，禁止写入任何可执行实现。
5. 本轮只允许在草稿区与账本区工作，不得把草稿迁入 `shared/contracts/sim-trading/`，不得写入 `services/`。
6. 草稿完成后，补写 TASK-0002 的自校验结果、P0 Token 申请清单，并回写 handoff 摘要，等待 Jay.S 提供实际 P0 Token。

交付标准：

1. `shared/contracts/drafts/sim-trading/` 下四份草稿已补全并自校验。
2. `docs/prompts/公共项目提示词.md` 已清理重复/过时状态，口径与当前执行阶段一致。
3. `docs/reviews/TASK-0002-review.md` 或 handoff 中已补充草稿自校验结论与 P0 Token 申请清单。
4. 未发生任何 `shared/contracts/sim-trading/` 或 `services/` 目录的正式写入。
